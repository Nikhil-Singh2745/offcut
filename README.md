# ─── OFFCUT ───

> **A highly sophisticated, award-winning, premium design portfolio for a studio that does not exist.**

---

## What is this?
A showcase of plain HTML, CSS, and JavaScript pushed past the point of sanity. I could have built a clean, minimal, sensible landing page. 

**I chose to build this instead.**

---

## Features

* **100% Organic Code**
  Zero frameworks, zero libraries, zero npm packages, and zero borrowed effects.
  
* **Interactive Paint Canvas**
  Because nothing says "enterprise-ready design agency" like letting users scribble all over the site with virtual ink.
  
* **An Identity Crisis**
  Six beautifully detailed client projects that eventually confess the copywriter gave up halfway through.
  
* **Quality Control**
  A wandering digital employee who is just as confused about this website as you are.
